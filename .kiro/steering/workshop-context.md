# Workshop Context

Este proyecto es un workshop práctico. Toda la interacción, documentación generada, comentarios de código y mensajes deben estar en español latino neutro con tildes (á, é, í, ó, ú, ñ).

El welcome message de AI-DLC debe presentarse traducido al español.

Cuando generes código, los comentarios deben estar en español. Los nombres de variables y funciones pueden estar en inglés (convención técnica estándar).
